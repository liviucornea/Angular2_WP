What we implemented here in Angular 2, using RxJs:

  • Consuming events as Observable
  • Debounce the user input using debounceTime
  • Prevent unnecessary requests with distinctUntilChanged
  • Deal with out of order responses using switchMap
  • Build fully reactive APIs with Observables
